[基本思路]
针对数据原始文件设计特征, 使用lgbt模型与xgb模型进行融合, 并使用打包方法对两个模型分别进行特征过滤,均使用最佳特征方案进行模型训练,最后进行特征融合.
[1.代码加载路径]
合并数据元数据文件,使用DataMerge()类, 文件夹中已经附带了集合好的元数据,直接使用即可.
[2.预测代码路径]
预测代码路径默认为当前文件夹路径,也可指定路径.
路径指定在 generate_csv_file_from_yTestB()中
[3.预测结果比对]
